## 📌 About the Project

A collection of advanced performance analysis and telemetry tools for Dead by Daylight. Each tool provides real-time tracking, aura reading, and gameplay improvement features.

## 🔧 Available Tools

- **ARCANE** — Sophisticated analyzer with smooth aim assistance, comprehensive ESP for survivors, killers, perks, generators, and hooks.
- **FECURITY** — Secure and powerful analyzer with smooth aim assistance, extensive ESP for auras, generators, pallets, hooks, and items.
- **NIGHTMARE** — Terrifying analyzer with deadly accurate aim assistance, deep ESP for auras, generators, hooks, and anti-stun features.
